# Proyecto-de-Aula-Electr-nica-digital-III-
Este repositorio contiene los avances del proyecto de aula para el curso de electrónica digital III

Sisifo.IO es un pequeño robot que permite clasificar tres objetos de diferentes colores. El usuario puede indicarle por medio del dispositivo móvil, cual de los tres objetos desea que el robot seleccione y mueva a una posición determinada.

- El robot tendrá una estructura motriz de dos o cuatro ruedas (Aún por definir) y podrá moverse en las cuatro direcciones.

- El robot tendrá un diseño tipo plataforma de carga (montacargas) para transportar el objeto seleccionado.

- El robot ejecutará los movimientos gracias a una unidad de medición inercial (IMU) y el microcontrolador. Dichos movimientos harán parte de la programación, por lo que no se necesitarán indicaciones en el entorno. Bastará con un espacio amplio y plano.  

- La identificación de los objetos se realizará por medio de la identificación del color empleando sensores de color.

Nos motiva realizar este proyecto porque nos llama la atención el campo de la automatización, la robótica y el control de procesos.
